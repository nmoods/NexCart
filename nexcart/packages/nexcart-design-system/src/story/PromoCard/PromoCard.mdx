# PromoCards

A reusable component that displays promotional product cards with:
- Background image
- Brand logo
- Product image
- Discount text


## Usage

```tsx
import { PromoCard } from "@nexcart/nexcart-design-system/PromoCardStories";

const MyComponent = () => (
  <PromoCard
        items={[
            {
             label: "Dhurandhar",
                     brand: "The revenge",
                     discount: "Tera baap",
                     image: Rehman,
                     logo: Rehman,
                     background: Dhurandhar,
              },
      ]}/>
);
