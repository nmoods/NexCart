# Product Header

A simple header component used to display a section title with a highlighted part and an optional "View All" action.


## Usage

```tsx
import { ProductHeader } from "@nexcart/nexcart-design-system/ProductHeaderStories";

const MyComponent = () => (
  <>
    <ProductHeader/>
  </>
);
```
