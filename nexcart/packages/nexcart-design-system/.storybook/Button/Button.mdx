# Button

The Button component is a foundational UI element used for triggering actions or navigation.

## Usage

To use the button, import it from the design system:

```tsx
import {Button} from "@nexcart/nexcart-design-system";

const MyComponent = () => (
  <>
    <Button>My button</Button>
    <Button variant="secondary">My button</Button>
  </>
);
```

## Props

- **variant**: `"primary" | "secondary"` - The visual style of the button (default: `undefined`)
- **loading**: `boolean` - Shows the loading state and disables the button
- **disabled**: `boolean` - Disables the button
- **children**: `ReactNode` - The content inside the button

All other standard HTML button attributes are supported and will be passed directly to the underlying `<button>` element.
